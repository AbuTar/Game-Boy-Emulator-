The Game Boy uses a Sharp LR35902 CPU

# Registers

The Game Boy has 8 8-bit General Purpose Registers as well as 2 16-bit registers

# Flags

F Register bits: Z N H C 0 0 0 0  Bits 0-3 always set to 0
                 │ │ │ │
                 │ │ │ └─ Carry (C): Did the result overflow?
                 │ │ └─── Half-Carry (H): Did bits 0-3 overflow into bit 4?
                 │ └───── Subtraction (N): Was this a subtraction?
                 └─────── Zero (Z): Is the result zero?


For a  half-carry, you use an AND (&) mask with the value being passed into the function and 0xF:
E.g.    0001 & 1111 (0xF)
        = 1 0000 - this is greater than 15 and so bit 3 overflowed into bit 4
        Code example: set_flag(cpu, FLAG_H, ((cpu->A & 0xF) + (value & 0xF)) > 0xF);
         - essentially set_flag(cpu, FLAG_H, (val in A + val in B > 15))
